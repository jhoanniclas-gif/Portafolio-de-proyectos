# Portafolio-de-proyectos
Este repositorio contiene ejercicios básicos de programación en Java realizados en Visual Studio Code.  
Cada ejercicio incluye:
- Representación visual (no-code) para entender la lógica paso a paso.  
- Código en Java para ejecutarlo en consola.  

## Lista de ejercicios
1. Operaciones básicas (suma, resta, multiplicación, división)  
2. Mayor o menor de edad  
3. Mayor de tres números  
... hasta el 20.  

## Objetivo
Documentar mi proceso de aprendizaje en Java y reforzar la lógica de programación.
